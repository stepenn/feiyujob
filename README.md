
本项目是新手初次开发微信小程序，期间踩了很多坑！

用到的插件包括：ECharts,wxPrase(解析html标签),以及公共搜索组件wxSearchView,

开发者过程中主要遇到的问题包括：onShow对页面的监听显示,tabBar页面跳转,下拉加载onPullDownRefresh,getLocation当前位置定位等问题
